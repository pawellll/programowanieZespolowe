Main page

Resource monitoring system

The site presents a system for monitoring computer resources such as CPU utilization and memory usage.
Measurements for individual computer resources are loaded using Sensors, then transferred to the Monitors and shared for clients. Provided measurements are used to create graphs and composite measurements.

Side menu consists of pages like:
  • Monitors - allows user to connect to Monitor whose measurements should be monitored
  • Hosts - contains a list of devices connected to Monitor
  • Resources - displays a list of monitored resources for specific host
  • Measurements - shows simple measurements loaded from Sensors for individual resource and composite measurements
  • Graphs - displays measurements as graphs
  • Measurements archive - stores archive of measurements
